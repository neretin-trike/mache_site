## Mache

Статический лендинг для тренировки вёрстки.

Используемые технологии: 
+ Bootstrap 4;
+ Sass;
+ Haml;
+ Gulp;
+ Bower

Необходимые инструменты для запуска в режиме разработки:
+ node.js
+ gulp(v3)
+ bower

### Запуск проекта в режиме разработки
`gulp`
